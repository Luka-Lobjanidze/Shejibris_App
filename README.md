აპლიკაცია წარმოადგენს Android პლატფორმისთვის შექმნილ შეჯიბრის ორგანიზების სისტემას,
რომელიც მომხმარებელს აძლევს შესაძლებლობას სწრაფად შეადგინოს მონაწილეთა სია,
იგი მარტივად შეძლებს გამარჯვებულის გამოვლენას.ასევე შესაძლებელია მონაწილეების დისკვალიფიკაცია.


აპლიკაციის შექმნისას გამოყენებულია:

Kotlin
Jetpack Compose
Material 3
MVVM Architecture
Room Database
Kotlin Coroutines
StateFlow
Navigation Compose
Kotlin Serialization


აპლიკაციის მუშაობის პრინციპი:
1.მომხმარებელი ამატებს მონაწილეებს.
2.მონაწილეები იყოფიან ჩვეულებრივ და განთესილ მონაწილეებად.
3.მონაწილეთა მონაცემები ინახება Room მონაცემთა ბაზაში.
4.შემდეგი მონაწილის დამატებისას სისტემა სთავაზობს უკვე არსებულ სახელებს (Autocomplete).
5.ტურნირის დაწყების შემდეგ მონაწილეები შემთხვევითად ნაწილდებიან.
6.თითოეულ ეტაპზე ირჩევა გამარჯვებული.
7.საბოლოოდ გამოვლინდება ტურნირის გამარჯვებული.
